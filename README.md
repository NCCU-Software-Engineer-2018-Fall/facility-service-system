## Installation

```
$ git clone https://github.com/NCCU-Software-Engineer-2018-Fall/facility-service-system.git

$ cd facility-service-system  

$ npm install 
```

## Usage
### Start & Watch 

```
$ npm start
```

### Build for production

```
$ npm run build
```
